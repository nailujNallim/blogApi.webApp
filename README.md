# Web App Angular (for API Consumption)

•	Running on: http://zmgblogwebapp.azurewebsites.net/

•	Software prerequisites
  o	Node JS
  o	Angular 8
  
•	Steps to build

  1.	Download the Project
  2.	Open it on Visual Studio Code
  3.	Install Node JS
  4.	Install Angular CLI
  5.	Exec npm install
  6.	Exec ng serve -o
